# food_delivery_app

A small simulation app for food ordering and delivery created with Dart and Flutter.

## Features

-   Log in / Sign up
-   Browse the menu
-   Add to cart
-   Input card details
-   Generate receipt

## Key Points

-   Firebase authentication and database
-   Provider state management
-   Packages: firebase, flutter_credit_card, collection
